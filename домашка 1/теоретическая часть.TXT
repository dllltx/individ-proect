Основные преимущества Git:
История изменений - Git запоминает все версии файлов, можно откатиться назад

Ветки - можно работать над разными задачами параллельно

Командная работа - несколько человек могут работать над одним проектом

Резервная копия - у каждого участника есть полная копия проекта

Что решает Git в команде:
Кто что сделал - видно автора каждого изменения

Конфликты - Git помогает объединять изменения от разных людей

Безопасность - если что-то сломалось, можно вернуть рабочую версию

Локальный и удаленный репозиторий:
Локальный - на твоем компьютере, работаешь один

Удаленный - на сервере, для командной работы

Проще говоря: Локальный репозиторий - твой личный черновик, удаленный - общая тетрадь для всей команды
